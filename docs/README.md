# Documentation

This folder holds the handbook itself. Content lives under [modules/](modules/), one folder per module (`00-foundations` … `15-capstone`), each containing lesson files named `NN.M-slug.md`.

See [../CURRICULUM.md](../CURRICULUM.md) for lesson outcomes, [../REPOSITORY_STRUCTURE.md](../REPOSITORY_STRUCTURE.md) for conventions, and [../templates/lesson-template.md](../templates/lesson-template.md) for the required lesson format.

> [!NOTE]
